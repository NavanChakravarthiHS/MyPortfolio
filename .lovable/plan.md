

# Navan Chakravarthi — Developer Portfolio

## Overview
A modern, premium developer portfolio with dark gradient UI, glassmorphism effects, smooth animations, and a recruiter-friendly layout.

## Design System
- **Theme**: Dark blue/purple gradient background with glassmorphism cards
- **Font**: Inter (already available) with clean typography scale
- **Colors**: Deep navy (#0a0a1a) → purple (#1a0533) gradients, cyan/purple accent glows
- **Border radius**: Rounded containers with soft glowing borders

## Sections

### 1. Fixed Navigation Bar
- Name/logo on the left, menu links (Home, About, Skills, Projects, Contact) on the right
- Glassmorphism background with backdrop blur
- Smooth scroll to sections on click, hover underline animations
- Scroll progress indicator bar at the top

### 2. Hero Section
- **Left**: "Hello, I'm Navan Chakravarthi" heading with gradient text, subtitle "Java Developer | Full Stack Developer", short bio, two CTA buttons (View Projects, Download Resume)
- **Right**: Profile image placeholder in a glowing rounded frame
- Floating geometric shapes (circles, triangles) with subtle CSS animations
- Fade-in and slide-up entrance animations

### 3. About Section
- Two-column layout: professional bio text + decorative image/illustration
- Scroll-triggered reveal animation
- Highlights learning journey in Java and Full Stack Development

### 4. Skills Section
- Grid of glassmorphism skill cards with icons
- Skills: Java, OOP, HTML, CSS, JavaScript, MongoDB, Git, GitHub
- Animated progress bars/indicators
- Hover glow and scale effects on cards

### 5. Projects Section
- Card grid with project: "Quiz Quest – Full Stack Quiz Application"
- Each card: project image, description, tech stack badges, GitHub + Live Demo buttons
- Hover elevation/scale animation with glow effect

### 6. Contact Section
- Clean glassmorphism contact form (Name, Email, Message fields)
- Social links: GitHub, LinkedIn, Email with hover animations
- Animated submit button with gradient

### 7. Animations & Effects
- Intersection Observer for scroll-triggered fade-in/slide-up animations
- Floating background shapes with CSS keyframe animations
- Subtle gradient movement in background
- Scroll progress indicator
- Button hover effects with glow

### 8. Responsive Design
- Fully responsive for mobile, tablet, and desktop
- Mobile hamburger menu for navigation
